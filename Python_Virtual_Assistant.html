Python Virtual Assistant

About This Project

This project is a desktop virtual assistant written in Python. It is activated and controlled entirely by voice commands, output is also given through voice synthesis. It makes use of numerous APIs from serices such as Wikipedia and YouTube.

The following is a list all capabilities which have been implemented thus far:

  • 'Repeat' command for echoing voice input given into the assistant.
  • 'Play' command which makes a call to the YouTube API to query for a song, the song is then played. (Every URL that has been played is saved in a file so repeated queries for the same song do not waste the API quota.)
  • 'Wikipedia' command that queries Wikipedia for the question that follows the command. The answer is then spoken with voice synthesis.
  • 'Execute' command which is able to run a custom shell script that was written by the user. This is a more advanced feature but also allows for easy programability and automation of desktop tasks using the voice assistant.

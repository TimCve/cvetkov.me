SystemZero

About This Project

SystemZero is a 32-bit computer operating system for the x86 CPU architecture, implemented in the C Programming Language, without the use of any external dependencies (apart from the BIOS firmware). The bootloader, kernel and all system utilities thus far have been written exclusively by me.

A basic overview of the features of SystemZero includes:

  • A multi-drive bootloader.
  • A scrolling text console driver.
  • Support for PS/2 keyboards.
  • Support for up to 4 IDE hard drives at once.
  • A read and write filesystem.
  • Fully featured set of utilities for interfacing with the filesystem.
  • Editors for hex code and ASCII text.
  • A basic scripting language interpreter.
  • Ability to switch terminal colour schemes.

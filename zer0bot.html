About This Project

Zer0bot is a bot made for the Discord chat program, written in the Discord.js module for Node.js. It is able to perform functions like greeting new users, moderating the chat and formatting messages in certain ways if a command is provided. It makes use of a file format managed with a text parsing program to manage multiple servers at once.

The bot is currently hosted on the same server as this website. It is directly synced with the git repository and is currently serving 3 separate Discord servers.

The image above is a section of a Discord chat, showing how interactions with zer0bot work. The bot gives a different list of commands depending on whether the user is an administrator or not. Administrators are able to interact with the bot settings to set parameters like the channel where the bot greets new users.

The administrator of every server can set the channel where the bot welcomes new users like so. Non-administrators are forbidden from running those commands.
